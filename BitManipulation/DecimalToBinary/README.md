# Decimal to Binary

Given a decimal number `n`, return its binary equivalent.

---

## Example 1:

**Input:**  
n = 12  
**Output:**  
1100  
**Explanation:**  
The binary representation of 12 is "1100", since 12 = 1×2³ + 1×2² + 0×2¹ + 0×2⁰.

---

## Example 2:

**Input:**  
n = 33  
**Output:**  
100001  
**Explanation:**  
The binary representation of 33 is "100001", since 33 = 1×2⁵ + 0×2⁴ + 0×2³ + 0×2² + 0×2¹ + 1×2⁰.

---

## Constraints:

- 1 ≤ n ≤ 2³¹ - 1
